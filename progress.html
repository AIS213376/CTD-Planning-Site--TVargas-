Fireworks VS. Dreamweaver

I used two programs, Fireworks and Dreamweaver, to create a prototype for my storyboard.

Here are the similarities and differences between the two programs.

Similarities Differences in FW Differences in DW
  • Both programs are very visual-oriented. They give you an outlook of what your site can look like.
  • Both are used fairly often for web design.
  • Final files can be opened to a web browser to experience the website.
  • FW is focused on production of Web graphics and the looks of the site rather than its usability.
  • FW is very quick to use to create a big picture of the website. With it's splicing tool, you can determine which parts of the webpage are usable and which are not.
  • The final FW file can be exported into an HTML and CSS code
  • DW is focused more on the structure and the coding of webpages/websites.
  • Provides a design mode where you can code and see what it looks like, so you get both ideas of what the code is and what the code looks like.
