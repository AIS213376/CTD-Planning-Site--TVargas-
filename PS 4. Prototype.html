EC/ES PE Website

Later, the wireframe will be transformed into a storyboard, where it will show a rough outlook of what the site would be. For the PE Website, I created the Storyboard using Fireworks.

Storyboard Thoughts

I focused more on what the flowchart was like and what the objectives were. I created this storyboard to reflect easy navigation for better user experience, and I designed it in a simplistic way so that the client can edit it in the future.

I showed this storyboard to another classmate (Rebecca). Here's what she has to say:

  • Audience problems
    • Will the teachers really use it?
  • Add a calendar to empty spaces
After considering what Rebecca has to say, I decided to change the three bars in the end of the page from navigation for students, teachers, and parents into a calendar, which is probably what the users would go to the website for (to check current events).

Planning Site

My wireframes for the planning site will then be used to create a storyboard. Since my website has a consistent look all throughout, I decided to create my storyboard from Dreamweaver.

In Dreamweaver, I can create a template that will be used for every webpage in my planning website.

All of the boxes in the template are AP Divs, which are divisors with absolute positioning in the file.

Everything but the two boxes next to the navigation are permanent, which means it can only be edited in the template. The rest are editable regions, which can be edited in any webpage.

I then edit the navigation buttons so that the user has a better navigation experience in the website.

To do this, I head to Adobe Fireworks and create buttons for navigation.

These navigation buttons were created in Fireworks using vectors and text. To create them into an interactive button, however, I have to convert the vectors into a symbol.

From there, I tweak the different states of the button (ie. unclicked, hovered, and clicked).

Afterwards, I export the button to Dreamweaver as a Fireworks HTM file.

I then create another AP Div for the Fireworks HTM file, and now the buttons are ready to use in my planning site (as seen in the navigation bar in the left).
